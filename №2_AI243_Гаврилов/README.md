# Лабораторна робота №2

Тема: реєстрація на сайті онлайн-уроків (Варіант 8).

## Що реалізовано

- Чиста HTML-сторінка українською мовою без CSS та JavaScript.
- Форма з усіма полями за умовою завдання:
  - ПІБ
  - Дата народження через `select`
  - E-mail
  - Країна (`select`, 10+ варіантів)
  - Місто (`radio`, 5+ варіантів, один `name`)
  - Мови (`checkbox`, 5+ варіантів, різні `name`)
  - Предмети (`select multiple`, 10+ варіантів)
  - Спосіб оплати (`select`, 10+ варіантів)
  - Номер банківської картки
  - Пароль
  - Додаткова інформація (`textarea`)
  - Приховане поле `student` зі значенням `Ваше ім'я`
  - Кнопки `submit` та `reset`

## Файл

- `index.html` — головна сторінка лабораторної.

## Як відкрити

Відкрийте `index.html` у будь-якому сучасному браузері.
