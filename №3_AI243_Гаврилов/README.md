# Лабораторна робота №3 - Основи CSS

Цей проект містить рішення для лабораторної роботи №3, яка охоплює основи CSS. Робота складається з двох основних завдань.

## Завдання 1: CSS селектори

У цьому завданні були застосовані різні CSS-селектори для стилізації HTML-документа `index.html` відповідно до наданих вимог.

Були використані наступні типи селекторів:
- Селектори за ID (`#age-table`)
- Селектори тегів у поєднанні з селекторами ID (`#age-table label`)
- Селектори псевдокласів (`:first-child`, `:last-of-type`, `:hover`, `:nth-child(odd)`, `:nth-child(even)`)
- Селектори атрибутів (`form[name="search-person"]`, `input[name='info[1]']`)
- Загальний селектор спорідненості (`~`)

Всі стилі для цього завдання знаходяться у файлі `style.css`.

## Завдання 2: Медіа-запити

Друге завдання полягало у створенні адаптивного дизайну за допомогою медіа-запитів. Три елементи `<div>` на сторінці змінюють свій вигляд залежно від ширини видимої області вікна браузера:

- **До 299px:** `div` мають зеленувато-жовтий фон, висоту 50px та відступ 20px.
- **Від 300px до 799px:** `div` стають темно-оранжевими, висотою 100px та з відступом 50px.
- **Більше 800px:** `div` набувають синьо-фіолетового кольору, висоти 200px та відступу 100px.

Це демонструє використання медіа-запитів для створення гнучких та адаптивних веб-сторінок.
