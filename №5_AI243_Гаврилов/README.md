# Лабораторна робота №5 (Варіант 8 — LinkedIn)

Реалізовано сторінку за блочною моделлю з елементами:
- `header` (логотип і назва компанії)
- `main` (зображення + опис компанії з різним обтіканням)
- `sidebar` (посилання History, Team, Contact з hover-ефектом)
- `footer` (ПІБ і дата)

## Структура стилів
Підключення виконано через `styles/imports.css`:
1. `normalize.css`
2. `variables.css`
3. `fonts.css`
4. `main.css`
5. `responsive.css`

## Запуск
Відкрийте `index.html` у браузері.

Або запустіть локальний сервер (опційно):

```powershell
cd "C:\Users\Alexander\OneDrive\Desktop\№5_AI243_Гаврилов"
python -m http.server 8000
```

Після цього відкрийте: `http://localhost:8000`
