# Лабораторна робота №4

Тема: **"Знайомство з JavaScript. Модель DOM"**  
Варіант: **8 (Спорт)**

## Структура проєкту

- `index.html` - розмітка сторінки
- `style.css` - оформлення інтерфейсу
- `script.js` - JavaScript-логіка (через `DOMContentLoaded`)
- `assets/1.jpg.webp` - перше зображення (футбол)
- `assets/basketball.svg` - друге зображення (баскетбол)

## Реалізовано за вимогами

1. Зміна зображення при кліку по картинці.
2. Кнопка зі зміною тексту: `Почати гру` / `Завершити гру`.
3. Кнопка для зміни тексту параграфа і `div` (спорт і рахунок).
4. Модальне вікно з відкриттям і закриттям.
5. Введення імені з перевіркою порожнього поля та персоналізованим привітанням.
6. Циклічна зміна фону сторінки (4 кольори).

### Додаткові завдання

- Закриття модального вікна клавішею `Esc`.
- Збереження імені в `localStorage` та відновлення після перезавантаження.
- Лічильник натискань кнопки зміни тексту.
- Плавні CSS-анімації для зміни зображення та модального вікна.
- Перемикач світлої/темної теми з збереженням у `localStorage`.

## Як запустити

1. Відкрити файл `index.html` у браузері.
2. Перевірити всі кнопки та сценарії взаємодії з DOM.
