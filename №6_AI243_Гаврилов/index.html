CSS-анімації

Анімації створюються через @keyframes та властивості animation-name, animation-duration, animation-timing-function і animation-iteration-count. На цій сторінці меню використовує циклічну пульсацію, а картки - ефект плавної появи.

Ефект: keyframes + easing

2D-трансформації

Для інтерактивності застосовано translate, rotate і scale. При наведенні пункти меню піднімаються, трохи повертаються і збільшуються, що підкреслює активність елементів.

Ефект: translate + rotate + scale

3D-трансформації

Для інформаційних блоків використано перспективу контейнера та обертання по осях X/Y. На ховер картка нахиляється в просторі, формуючи об'ємний ефект без JavaScript.

Ефект: perspective + rotateX/rotateY

Практична частина

Центральна зона побудована у дві колонки через CSS Grid. Кожен блок має власну затримку появи, що створює каскадну анімацію. Фоновий діагональний відблиск рухається окремо для візуальної динаміки.

Ефект: grid + stagger + decorative motion

Висновок

У роботі реалізовано вимоги: меню в шапці, кілька інформаційних блоків у дві колонки, різні CSS-анімації та комбінації 2D/3D-трансформацій. Верстка адаптивна й охайна для демонстрації в лабораторній.

Готово до перевірки

Додатково: доступність

Підтримано режим prefers-reduced-motion, у якому анімації вимикаються. Це зменшує навантаження та робить сторінку комфортнішою для користувачів із чутливістю до руху.

Ефект: reduced-motion support
