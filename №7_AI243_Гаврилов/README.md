# ЛР7 — Валідація форми (варіант 8)

Студент: **Гаврилов Олександр В`ячеславович**  
Тема: валідація даних форми за допомогою CSS та JavaScript.  
Варіант 8: **форма підписки на освітню платформу**.

## Що реалізовано

- Поля форми: ім'я, електронна адреса, пароль, кнопка надсилання.
- Валідація в реальному часі (`input`) і при відправці (`submit`).
- Правила перевірки:
  - ім'я: мінімум 2 символи, лише літери/пробіли/апостроф/дефіс;
  - email: коректний формат адреси;
  - пароль: мінімум 8 символів, щонайменше одна літера і одна цифра.
- Візуальна індикація через CSS-класи `valid` та `invalid`.

## Запуск

Відкрийте файл `index.html` у будь-якому сучасному браузері.

## Структура

- `index.html` — HTML-розмітка форми.
- `styles/main.css` — усі стилі сторінки.
- `scripts/main.js` — JavaScript-логіка валідації.
